Remember, if you make changes to _config.yml, you need to restart the Jekyll server:
	1	Stop the server by pressing Ctrl+C in the terminal.
	2	Start the server again with the command:jekyll serve -l -H localhost
		This approach ensures that you are working within the correct context for each Jekyll project.
